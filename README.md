# RxAPI
> HTTP API & WEB SOCKETS SERVER
>
> https://warlock.gitbooks.io/rxapi

## Install
```sh
sudo npm i rxapi -g
```

## Install from npm
```sh
sudo npm i rxapi -g
```

## Make new api project:
``sh
rxapi new project
cd project
rxapi run
```

## Generate a simple dummy API Restfull in memory for testing:

``sh
rxapi dummy project
cd project
rxapi run
``sh
```
